Amazon Teleporter Extension


This is a simple Firefox extension that flips between Amazon.com and Amazon.co.uk. See this blog post for more info. For most books and CDs, this will let you jump back and forth between the UK page and the US/.com page. Why would you do this?

  • Access extra reviews For whatever reason, Amazon doesn't consolidate all the English-language reviews. This leaves Amazon.co.uk quite bare, and Amazon.com users might also benefit from a different set of opinions, coming predominately from UK/European users.
  • Check inventory So you have a deadline on Monday and Amazon's not stocking that all-critical book? Order it from abroad, maybe the other website has a copy!
  • Access extra features Amazon.co.uk seems to lag behind with some features (such as searching inside books, previews, author blogs) ... this is an easy way for those users to access those features for a given book.

Domain Teleporter

This is a Greasemonkey script that does the same as the Amazon Teleporter Extension above. However, it's not limited to Amazon; by opening up the Greasemokney dialog's settings for this script, you can make it apply to all sites (specify "*" for Included Pages, instead of the default Amazon settings). You need Greasemonkey installed to make this script work.

domainteleporter v0.1 (Greasemonkey script)

This is an Experiment
