IFrame Loader - A JQuery Plugin

This is JQuery IFrame Loader, a plugin to help you load websites into iframes, by Michael Mahemoff, under MIT License. The main benefit is you know when the iframe has fully loaded, and there are some other features shown below.

Download
iframe.jquery.js

Usage

$("iframe").src("http://google.com"); $("iframe").src("http://google.com", function(iframe, duration) { alert("That took " + duration + " millis."); }); $("iframe").src("http://google.com", function(iframe, duration) { alert("That took " + duration + " millis."); }, { timeout: 1000, onTimeout(iframe, duration) { alert("we'll never know!"); } }); $("iframe").src("http://google.com", function(iframe, duration) { alert("That took " + duration + " millis."); }, { onReady(iframe, duration) { alert("we'll never know!"); } });
